Detailed Description

The aim of this class is to manage different Timer created previously.

This class can not be used without defining Timers Previously.

Attention
- You can use this class for manager one or more timers
- If you want to manage more than one Timer at a time you must specify a Timer pool
- A Timer pool is a Timer array
